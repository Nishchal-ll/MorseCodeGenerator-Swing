# Morse Code Generator

A simple **Java Swing application** that allows users to convert text into Morse code. The application features a clean GUI with an icon and project title.

![morse](https://github.com/Nishchal-ll/MorseCodeGenerator-Swing/blob/000eb7c8d849d090f83b2df9b4fb9e4825193aca/Morsee.png)
---

## Features
- User-friendly **GUI** using Java Swing
- Displays an **icon and project title**
- Ready to extend for **text-to-Morse code translation**

---

## Requirements
- Java JDK 8 or above
- IDE such as IntelliJ IDEA, Eclipse, or NetBeans
